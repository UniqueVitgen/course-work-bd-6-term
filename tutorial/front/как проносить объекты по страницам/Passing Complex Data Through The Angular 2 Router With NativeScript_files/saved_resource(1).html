                                                            Nic Raboy Mod braayim • a year ago

                                                            Per the Angular documentation you probably want to use the replaceUrl parameter:

                                                            https://angular.io/docs/ts/...
                                                            https://angular.io/docs/ts/...

                                                            From what I understand, you'd do something like this:

                                                            this.router.navigate(['page'], { replaceUrl: true });

                                                            In theory it will make the new page the top of the stack, so trying to navigate back will only close the application. You might also look at NativeScript RouterExtensions as well:

                                                            https://docs.nativescript.o...
